Project Overview

The Geo-Information Data Visualization Project is a full-stack wildfire risk platform developed as a CSUN Computer Science Senior Design capstone. The system ingests geospatial data — including weather, vegetation, elevation, and historical fire records — and feeds it into a trained machine learning model to generate real-time risk predictions displayed on an interactive map.

The platform supports residents, researchers, and emergency planners who need to understand where wildfire risk is highest across California. It provides a React-based frontend with GIS layer controls and date filters, a Flask backend with a live prediction API, user-defined threshold alerts, and an admin panel for managing data refresh schedules.

Major Features

  • Interactive Geospatial Maps

    Dynamic GIS layers with date filters let users explore wildfire risk zones, affected regions, and location-based environmental patterns across California in real time.

  • ML Risk Prediction

    A trained scikit-learn model provides single and batch wildfire risk predictions via REST API, powered by ingested weather, vegetation, elevation, and fire-history data.

  • Alerts & Notifications

    Users define custom risk thresholds and receive alerts when fire risk conditions meet their configured levels, keeping residents and planners proactively informed.

  • Data Ingestion Pipeline

    An automated ETL pipeline ingests and preprocesses geospatial data from weather feeds, vegetation indices, elevation models, and historical fire records on a configurable schedule.

  • Data Visualization & Analytics

    Visual overlays, risk dashboards, and analytical displays make complex environmental data accessible to both technical researchers and general users.

  • Admin Workflows

    Administrators configure data refresh schedules, manage user roles, and control system settings through a dedicated admin panel backed by secure API routes.
